# rest_API

This is an example of rest API.

The API is created using Spring boot. It supports following operations:
1. GET: /topics
   (returns all topics)
2. GET: /topics/id
   (return a topic only)
4. POST: /topics
   (create a topic)
5. PUT: /topics/id
   (update a topic)
6. DELETE: /topics/id
   (delete a topic)

The API can be tested with Postman or any other tools available.
